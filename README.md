# Glass_Type

<b> This project is carried out on dataset avaliable on Kaggle</b>

<b> Features in this dataset include: </b>
* Rl   (Lawrencium)
* Na   (Sodium)
* Mg   (Magnesium)
* Al   (Aluminium)
* Si   (Silicon)
* K    (Potassium)
* Ca   (Calcium)
* Ba   (Barium)
* Fi   (Iron)
* Type (Target variable)

<b> The Goal: </b>

To predict the type of glass based on the fatures above
